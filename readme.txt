===================================================================
RiskMap - Система управления рисками и моделирования угроз
Инструкция по установке и запуску (Windows)
===================================================================

ТРЕБОВАНИЯ:
- Python 3.10 или новее
- Установленный Git (желательно)

===================================================================
ШАГ 1: УСТАНОВКА
===================================================================

1. Скачайте проект или склонируйте репозиторий в удобную папку.
   Если это архив - распакуйте его.

2. Откройте эту папку в терминале (PowerShell или CMD).
   (В папке должен лежать файл manage.py)

3. Создайте виртуальное окружение (чтобы не засорять систему):
   Команда: python -m venv venv

4. Активируйте виртуальное окружение:
   Команда: .\venv\Scripts\activate
   (Если появится (venv) в начале строки - всё ок)

5. Установите необходимые библиотеки:
   Команда: pip install -r requirements.txt

===================================================================
ШАГ 2: НАСТРОЙКА БАЗЫ ДАННЫХ
===================================================================

1. Создайте структуру базы данных:
   Команда: python manage.py migrate

2. Загрузите базу знаний уязвимостей (ВАЖНО для работы автосканера):
   Команда: python manage.py load_bank_templates
   
   *Если команда выдает ошибку, выполните загрузку через Shell:*
   1. python manage.py shell
   2. Скопируйте код из файла docs/init_db_script.py (если он есть) или создайте шаблоны вручную через админку.

3. Создайте администратора (суперпользователя):
   Команда: python manage.py createsuperuser
   (Введите логин и пароль, email можно пропустить)

===================================================================
ШАГ 3: ЗАПУСК
===================================================================

1. Запустите сервер:
   Команда: python manage.py runserver

2. Откройте браузер по адресу:
   http://127.0.0.1:8000/

===================================================================
КАК ПОЛЬЗОВАТЬСЯ (БЫСТРЫЙ СТАРТ)
===================================================================

1. Войдите под созданным логином/паролем.
2. Нажмите "Создать процесс" (например, "Регистрация клиента").
3. Добавьте шаги процесса. 
   ВАЖНО: Для проверки авто-сканера используйте в названиях шагов 
   ключевые слова, например:
   - "Ввод email"
   - "Загрузка файла"
   - "Админка"
   - "Переход по ссылке"
4. Нажмите кнопку "Авто-подбор" (оранжевая кнопка).
5. Система автоматически найдет потенциальные угрозы и предложит меры защиты.
6. В разделе "Аналитика" можно посмотреть статистику по рискам.

===================================================================
ВОЗМОЖНЫЕ ПРОБЛЕМЫ
===================================================================
- Если пишет "Script is disabled...", введите в PowerShell:
  Set-ExecutionPolicy RemoteSigned -Scope Process

- Если не находит manage.py, убедитесь, что вы находитесь в корневой папке проекта.
